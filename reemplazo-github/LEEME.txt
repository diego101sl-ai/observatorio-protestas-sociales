CÓMO SUBIR EL NUEVO DISEÑO A GITHUB
====================================

Este zip contiene solo lo que cambia:

  index.html
  css/style.css
  js/app.js

(js/sources/gdelt.js NO cambió; data/ y vendor/ de tu repo no se tocan.)

Pasos (desde github.com, sin instalar nada):

1. Abre tu repositorio: diego101sl-ai/observatorio-protestas-sociales
2. Pulsa «Add file» → «Upload files».
3. Arrastra a la vez: index.html, la carpeta css y la carpeta js
   (GitHub respeta las rutas y reemplaza los archivos existentes).
4. Abajo escribe un mensaje, p. ej. «Rediseño institucional-documental»,
   y pulsa «Commit changes».
5. Espera 1–2 minutos a que GitHub Pages publique y recarga tu web
   con Ctrl+F5 (o Cmd+Shift+R en Mac) para saltarte la caché.

FOTO DE PORTADA (opcional, recomendada)
---------------------------------------
Sube una fotografía con derechos de uso (p. ej. Wikimedia Commons,
licencia CC) como:  img/hero.jpg   (carpeta img en la raíz del repo)

- Formato JPG, apaisada, mínimo ~1600 px de ancho.
- El duotono azul marino y el movimiento se aplican solos.
- Mientras no exista, se muestra el fondo gráfico de respaldo.
- Recuerda acreditar al autor si la licencia lo pide (puede ir en el pie).
